BioCenicana: Manual de Usuario

Guía Metodológica y de Ejecución de la Suite

Bienvenido a la Suite BioCenicana

Este manual te enseñará a usar las herramientas para analizar el ADN de tus plantas. Cada sección incluye la teoría científica y los pasos exactos para ejecutar el programa.

1. Filtrado de Calidad (vcf-filter)

¿Para qué sirve? Para limpiar tu archivo VCF. Quita los marcadores que tienen muchos datos faltantes o que no varían lo suficiente.

java -jar biocenicana.jar vcf-filter \
  -v mis_datos.vcf \
  -o datos_limpios.vcf \
  --min-maf 0.05 --max-missing 0.2
🚀 ¿Qué pasa paso a paso?
  • El programa abre tu VCF y lee cada SNP.
  • Calcula cuántos datos faltan. Si falta más del 20% (--max-missing 0.2), lo elimina.
  • Calcula si el rasgo es común. Si aparece en menos del 5% (--min-maf 0.05), lo elimina por ser muy raro.
  • Crea un nuevo archivo limpio llamado datos_limpios.vcf.

2. Estructura Poblacional (pop-structure)

¿Para qué sirve? Para agrupar las plantas por familias y ver si hay mezclas genéticas. Genera el dashboard interactivo.

java -jar biocenicana.jar pop-structure \
  -v datos_limpios.vcf \
  -p 10 -o mi_analisis
🚀 ¿Qué pasa paso a paso?
  • Cálculo de Dosis: Estima cuántas copias de cada gen tiene la planta (Ploidía 10).
  • PCA: Crea un mapa de puntos donde las plantas parecidas están juntas.
  • GMM: Calcula qué porcentaje de cada "raza" tiene cada planta (Gráfico de barras).
  • Kinship: Genera la matriz de parentesco de VanRaden.
  • Reporte: Crea mi_analisis.pca.html para que lo abras en tu navegador.

3. Desequilibrio de Ligamiento (ld-decay)

¿Para qué sirve? Para saber qué tan rápido se pierde la relación entre genes a medida que se alejan en el cromosoma.

java -jar biocenicana.jar ld-decay \
  -v datos_limpios.vcf \
  -o mi_ld_decay --max-dist 100000
🚀 ¿Qué pasa paso a paso?
  • Toma pares de genes y mide qué tan "conectados" están (r²).
  • Repite esto para miles de pares a diferentes distancias.
  • Calcula el promedio de conexión y genera un gráfico que muestra cómo cae esa conexión al alejarse.
  • Te entrega mi_ld_decay.html con la curva de decaimiento.

4. Estadísticas del VCF (vcf-stats)

¿Para qué sirve? Para obtener un resumen general de la salud y diversidad de tu base de datos.

java -jar biocenicana.jar vcf-stats -v mis_datos.vcf -o reporte_stats.tsv
🚀 ¿Qué pasa paso a paso?
  • Cuenta cuántos SNPs hay en total.
  • Calcula la Heterocigosidad (qué tan diversa es la planta internamente).
  • Calcula el PIC (qué tan informativo es cada marcador).
  • Genera una tabla reporte_stats.tsv lista para abrir en Excel.
