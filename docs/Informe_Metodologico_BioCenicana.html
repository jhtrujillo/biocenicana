Manual de Operación BioCenicana

Guía de Insumos, Procesos y Resultados Genómicos

1. Control de Calidad Genómico (vcf-filter)

📥 Entrada (Insumos)

  • Archivo VCF crudo: El archivo directo de secuenciación o llamado de variantes.
  • Parámetros: MAF mínimo (ej: 0.05) y porcentaje máximo de datos perdidos (ej: 0.20).

📤 Salida (Resultados)

  • Archivo VCF filtrado: Un archivo más pequeño y limpio, listo para análisis estadístico.
  • Resumen en consola: Conteo de cuántos SNPs fueron eliminados y cuántos sobrevivieron.
📌 Introducción: La limpieza de datos asegura que no estemos analizando "ruido" técnico como si fuera variación genética real.
💻 Explicación Bioinformática: El algoritmo revisa la calidad de cada sitio polimórfico y lo descarta si no cumple con los estándares poblacionales requeridos.
java -jar biocenicana.jar vcf-filter -v crudo.vcf -o filtrado.vcf --min-maf 0.05

2. Estructura Poblacional (pop-structure)

📥 Entrada (Insumos)

  • Archivo VCF filtrado: Datos limpios del paso anterior.
  • Ploidía: Nivel de copias cromosómicas (ej: 10 para caña).
  • PCs: Número de componentes a calcular (típicamente 10).

📤 Salida (Resultados)

  • Dashboard (.html): Reporte interactivo con gráficos PCA 3D, DAPC y Ancestría.
  • Matriz PCA (.csv): Coordenadas exactas de cada planta para graficar en R o Excel.
  • Matriz de Kinship (.csv): La relación de parentesco genómico de VanRaden.
📌 Introducción: Permite visualizar cómo se agrupan las plantas por su parecido genético global.
java -jar biocenicana.jar pop-structure -v filtrado.vcf -p 10 -o mi_analisis

3. Desequilibrio de Ligamiento (ld-decay)

📥 Entrada (Insumos)

  • Archivo VCF: Marcadores posicionados en el genoma (con coordenadas CHROM y POS).
  • Ventana Máxima: Distancia máxima en pares de bases para comparar SNPs (ej: 100kb).

📤 Salida (Resultados)

  • Gráfico de Decaimiento (.html): Curva interactiva que muestra cómo cae el LD con la distancia.
  • Tabla de Datos (.tsv): Los valores promedio de \(r^2\) por cada intervalo de distancia.
java -jar biocenicana.jar ld-decay -v filtrado.vcf -o mi_ld_decay

4. Estadísticas y Diversidad (vcf-stats)

📥 Entrada (Insumos)

  • Cualquier archivo VCF: Para obtener un diagnóstico rápido.

📤 Salida (Resultados)

  • Tabla de Resumen (.tsv): Valores de Heterocigosidad, PIC, MAF medio y calidad por muestra.
💡 Analogía Simple: Es como el recibo de caja de un supermercado: te dice exactamente qué compraste (cuántos SNPs), cuánto costó (calidad) y qué tan variada fue la compra (diversidad).
